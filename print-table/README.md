# Print data in table format

### Table design concepts

```
First table:

   Fruits   People   Animals
----------+--------+---------
   apples    Alice      dogs
  oranges      Bob      cats
 cherries    Carol     moose
   banana    David     goose



Second table
============

  Fruits | People | Animals
---------+--------+---------
   apples   Alice     dogs
  oranges     Bob     cats
 cherries   Carol    moose
   banana   David    goose



Third table
-----------

   Fruits | People | Animals
=============================
   apples |  Alice |    dogs
----------+--------+---------
  oranges |    Bob |    cats
----------+--------+---------
 cherries |  Carol |   moose
----------+--------+---------
   banana |  David |   goose
   
```
